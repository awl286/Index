Table Notes
  1. make a table using "table" and change related attributes within
  2. use tr to start a "row" or spacial boxes from left to right
  3. to put things in the rows and its boxes, use td (data)
    • this includes putting anything into those "boxes", such as tables as well
  4. thead just makes a bolded header at the top of the table


Creating HTML Tables
  • 1
  • create a new table row by using "tr"
  • use "td" to fill in the contents of the cells, going from left to right
  • "tr" = table row, "td" = table data
  • tables also have header, body, footer
    • thead = th
    • tbody = tb
    • tfoot = tf
